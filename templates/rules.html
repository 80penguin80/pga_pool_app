Competition Rules

  • Rule 1: All users must register with a valid email address.
  • Rule 2: Each user is allowed only one entry per tournament.
  • Rule 3: Picks must be submitted before the start of each tournament.
  • Rule 4: Users can only pick a golfer once per season.
  • Rule 5: Prize money is awarded based on the performance of selected golfers.
  • Rule 6: Any form of cheating or manipulation will result in disqualification.
  • Rule 7: The leaderboard is updated weekly based on the tournament results.
  • Rule 8: The decision of the organizers is final in all matters.
